FAQs: Frequently Asked Questions

Q: How do I know if a Citi Trends coupon code is valid?

A: The validity of a code is determined by its expiration date and any associated restrictions. Check the terms and conditions of the code before applying it at checkout.

Q: Can I use multiple Citi Trends coupon codes at once?

A: Typically, you can only use one coupon code per purchase. However, some codes might be combinable with other promotions. Check the specific terms and conditions of the code.

Q: What if my Citi Trends coupon code doesn’t work?

A: Ensure you have entered the code correctly, including capitalization and punctuation. If the code still doesn’t work, it might be expired or have specific restrictions that you don’t meet.

Q: How do I get notified about new Citi Trends coupon codes?
